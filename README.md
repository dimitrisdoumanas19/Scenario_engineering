# SEGNAR — Domain-Independent Narrative Ontology

Permanent identifier: **https://w3id.org/segnar/ontology**

This directory configures the w3id.org permanent redirect for the SEGNAR ontology. Requests to
`https://w3id.org/segnar/ontology` are content-negotiated:

- browsers / `text/html` → the HTML documentation,
- `text/turtle`, `application/rdf+xml`, `application/ld+json` → the ontology file (`SEGNAR_ontology.ttl`).

The ontology and its documentation are hosted at:
**https://github.com/dimitrisdoumanas19/Scenario_engineering** (served via GitHub Pages).

## About

SEGNAR (Scenario Engineering with Knowledge Graphs and Narrative-Aware Reasoning) is a
domain-independent ontology capturing universal narrative structures for scenario engineering in
high-stakes decision-making training. It defines eight core classes (Actor, Goal, Resource,
Event, Risk, Constraint, Value, Outcome) and the relationships that encode causal dependencies,
resource requirements, and goal and value conflicts. Domain-specific knowledge graphs instantiate
these classes for particular operational domains such as Search-and-Rescue, Border Security, and
Crisis Negotiation.
